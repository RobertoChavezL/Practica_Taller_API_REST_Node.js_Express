# API REST con Node.js y Express

Este proyecto forma parte de la práctica del taller de desarrollo con Node.js.  
El objetivo principal es construir una **API REST básica**, entender cómo funciona Express y aprender a manejar rutas, solicitudes HTTP y respuestas en formato JSON.

---

## Tecnologías utilizadas

- Node.js  
- Express  
- JavaScript  
- NPM (Node Package Manager)

---

##Estructura del proyecto

```
Práctica_Taller_API_REST_Node.js_Express/
├─ index.js
├─ package.json
├─ package-lock.json
├─ step-by-step.txt
├─ .gitignore
└─ routes/
└─ users.routes.js
```
---

## Cómo ejecutar el servidor

1. Instala las dependencias:
```bash
npm install

Inicia el servidor:

node index.js

Accede en tu navegador o Postman:

http://localhost:3000

```
